## 技巧

##### hugo server 前杀死占用的进程

新建文件  `hugo-server.sh`

```bash
lsof -ti:1313 | xargs kill
hugo server
```

`chmod +x hugo-server.sh`



配置 zshrc , 添加

`alias hugo-server="~/文件路径/hugo-start.sh"`
